# SLP Microsystems Website

Standalone marketing website for SLP Microsystems — Oracle Cloud IT Business & Consulting (Oracle Service Partner since 1999).

## Run locally

```bash
cd c:\Users\lmann\slpwebsite
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000).

## Build for production

```bash
npm run build
npm start
```

## Stack

- Next.js 14
- React 18
- Tailwind CSS
